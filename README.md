# 📚 프로젝트 소개  
   - 본 프로젝트는 포트폴리오 공유사이트 **'Valley-Tales'** 입니다.    
  
  
# 프로젝트 기간  
  -2024년 6월 17일(월) ~ 2023년 6월 28일(금), 약 2주 간  
  
  
# 프로젝트 목적
  -포트폴리오를 저장하고 공유하는 서비스를 제작한다.
  -수상내역, 자격증, 학력, 프로젝트 정보를 생성, 수정, 삭제할 수 있고 이를 다른 유저와 공유한다.
  -vanila JS로 CRUD를 구현한다.
